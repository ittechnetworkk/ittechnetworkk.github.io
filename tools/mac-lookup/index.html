About MAC Addresses

MAC Address: A unique identifier assigned to network interfaces for communications on the physical network segment.
OUI: The first 3 octets (24 bits) identify the organization that manufactured the network interface.
Unicast vs Multicast: The first bit determines if the address is for a single device (unicast) or multiple devices (multicast).
Local vs Global: The second bit indicates if the address is locally administered or globally unique.
